🎁 PM Pro Tips: Customize AI Tools Like a Boss

Most people use AI like Google. Here's how to go next-level — and actually get insightful, actionable results.

🧠 ChatGPT / Claude

  • Set a role: “You are a certified PMP helping draft a communication plan…”
  • Give examples: “Here’s how we handled this last quarter…”
  • Iterate: Ask for rewrites, simplification, or stakeholder-friendly versions

📝 Otter.ai / Fireflies.ai

  • Name meetings clearly: “Stakeholder Kickoff – Sept 15”
  • Tag speakers + decisions: Helps AI extract clean action items
  • Export clean text: Paste into AIpairwise for instant summaries

📂 Notion / Asana

  • Paste AI results: into project docs, briefs, tasks
  • Break it down: into subtasks and deadlines
  • Tag team members: and assign deliverables
