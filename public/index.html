🎁 PM Pro Tips: Customize AI Tools Like a Boss

Stop using AI like a search engine. Here’s how to get smarter results.

🧠 ChatGPT / Claude

  • Go to Settings → Custom Instructions: Fill in “What should the assistant know about you?” and “How should it respond?” so AI learns your style.
  • Use Memory features: Save your usual context, tone, preferences so you don’t repeat them each time.
  • Provide examples: “Use bullets for action items” or “Give me summary first, then details.”

📝 Otter.ai / Fireflies

  • Name your meeting clearly (“Stakeholder Kickoff – Sept 15”)
  • Mark decisions & actions as you speak so transcripts reflect them
  • Export transcript, clean it up, then feed into AI for summary

📂 Notion / Asana

  • Paste outputs into your project docs or task boards
  • Break them into subtasks, assign, add due dates
  • Review with your team so nothing slips
