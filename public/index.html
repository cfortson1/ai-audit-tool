🎁 PM Pro Tips: Customize AI Tools Like a Boss

Go beyond prompts — set up tools so AI works *with you*, not just for you.

🧠 ChatGPT / Claude

  • Go to Settings → Custom Instructions: Fill in “What should the assistant know about you?” and “How should it respond?” so your tone, goals, preferences stick.
  • Use Memory or Role features: Save your project style, domain terms, stakeholder names.
  • Supply examples: “Always list action items”, “Give me a summary then bullets”, etc.

📝 Otter.ai / Fireflies

  • Name meetings clearly so context is captured (“Stakeholder Kickoff – Sept 15”).
  • Use speaker tags + mark decisions during the meeting.
  • Export transcript, clean up, then feed into AI for summary.

📂 Notion / Asana

  • Paste AI draft outputs directly into project briefs/task boards.
  • Break them into subtasks with deadlines, owners.
  • Assign, tag team for review so nothing falls through.
